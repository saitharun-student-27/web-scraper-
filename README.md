# 🔍 PRO WEB SCRAPER

A web-based application that extracts and analyzes links from a webpage.

## 🚀 Features
- Extracts all links from a webpage
- Classifies internal and external links
- Detects broken links
- Displays results with charts
- Export data as CSV

## 🛠 Tech Stack
- Python (Flask)
- BeautifulSoup
- Requests
- HTML, CSS, JavaScript
- Chart.js

## ▶️ How to Run
```bash
pip install -r requirements.txt
python app.py
```

Open browser:
```
http://127.0.0.1:5000
```
